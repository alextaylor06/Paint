# Paint
Paint project CS250
An imitation of MS Paint using javafx

Feaures-  Shapes: Free Draw, Circle, Ellipse, Rectangle, Triangle, Line, Text
File: Saving, Opening images, Smart Exit                                                                                                       
Edit: Undo, Redo                                                                                                                            
Extra: Color Picker, color dropper, text size, font selector
